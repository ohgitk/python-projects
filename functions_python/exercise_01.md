## 1. Basic Calculator

Create four separate functions for basic mathematical operations:

- `add(num1, num2)` → returns the sum  
- `subtract(num1, num2)` → returns the subtraction  
- `multiply(num1, num2)` → returns the multiplication  
- `divide(num1, num2)` → returns the division

## 2. Personalized Greeting

Create a function that receives a name (a string) as a parameter and prints a welcome message to the screen, such as:

> "Hello, [name]! Welcome!"

In the main program:
- Ask the user for their name  
- Pass it to the function to display the greeting
